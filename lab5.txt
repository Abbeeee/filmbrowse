They are used to make it easier for various systems to communicate and send data to 
each other in ways that is understandable for them. It also makes it so that a user 
can access parts easier through the interface. An real life scenario is where you can 
use the google maps api to retrieve exact coordinates for cities.

    curl_init();

    $url = "https://catfact.ninja/fact?max_length=50";

    curl_setopt($curlconn, CURLOPT_RETURNTRANSFER, true);
